# ComfyUI-Text

## Install
```
cd /custom_nodes
git clone https://github.com/LZpenguin/ComfyUI-Text.git
```

## Add Font Files
There are 2 font files (en & cn) in this repo, u can test with them
```
cd ComfyUI-Text
mv ./font ../../models
```
